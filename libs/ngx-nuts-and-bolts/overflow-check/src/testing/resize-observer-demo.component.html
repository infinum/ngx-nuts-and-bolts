infOverflowCheck directive allows you to check whether the element is overflowing or not. Attach the directive, get the reference to it and check the $isOverflowing() signal value. Use this to add some styling or behavior depending on whether the element is overflowing or not.

In this example, overflowing elements will be set a class that makes the color red. Additionally, title attribute will be set, so that on cursor hover you see the whole text. You could use this to conditionally show some nicer tooltip/popover.
